## Overview

A demonstration e-commerce platform built to showcase modern frontend development practices and Shopify API integration. This project serves as a technical showcase of building scalable, performant e-commerce experiences with React and Next.js.

## Key Features

- **Full E-commerce Functionality**: Product browsing, filtering, cart management, and checkout
- **Shopify Backend**: Leverages Shopify's robust e-commerce infrastructure via Storefront API
- **Responsive Design**: Mobile-first approach ensuring excellent UX across all devices
- **Smooth Animations**: Framer Motion animations for polished user interactions
- **Real-time Updates**: Live inventory and pricing updates from Shopify

## Technologies Used

- **Next.js 14**: Latest features including App Router and Server Components
- **React 18**: Modern React patterns with hooks and context
- **Tailwind CSS**: Rapid UI development with utility classes
- **Framer Motion**: Production-ready animations
- **Shopify Storefront API**: Headless commerce backend
- **Vercel**: Deployment with edge functions and caching

## Technical Implementation

### Headless Commerce Architecture
The demo uses a headless commerce approach where:
- **Frontend**: Next.js handles all UI rendering and user interactions
- **Backend**: Shopify manages products, inventory, orders, and payments
- **API Layer**: GraphQL queries to Shopify Storefront API

### Key Technical Features
- Server-side rendering for SEO and performance
- Optimistic UI updates for cart operations
- Persistent cart state with local storage
- Dynamic product filtering and search
